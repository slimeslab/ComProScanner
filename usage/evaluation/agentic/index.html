    Agentic Evaluation¶

    This method leverages the reasoning capabilities of LLMs to evaluate extraction quality with human-like understanding.

    Basic Usage¶

    from comproscanner import evaluate_agentic

results = evaluate_agentic(
    ground_truth_file="ground_truth.json",
    test_data_file="extracted_results.json",
    output_file="agentic_eval.json"
)

    Parameters¶

    Required Parameters¶

    ground_truth_file (str)¶

    Path to ground truth data created by human experts.

    test_data_file (str)¶

    Path to extracted results to be evaluated.

    Optional Parameters¶

    weights (dict)¶

    Dictionary specifying weights for each metric during evaluation for ensuring scoring based on importance. The total should sum to 1.0.

    output_file (str)¶

    Path to save the evaluation results.

    extraction_agent_model_name (str)¶

    Name of the LLM model used for data extraction (e.g., "gpt-4o-mini").

    is_synthesis_evaluation (bool)¶

    Whether to evaluate synthesis-related information.

    llm (LLM)¶

    An instance of the LiteLLM class. Read more about LiteLLM instance from CrewAI here.

    Default Values

    weights = { "compositions_property_values": 0.3, "property_unit": 0.1, "family": 0.1, "method": 0.1, "precursors": 0.15, "characterization_techniques": 0.15, "steps": 0.1 }
    output_file = "agentic_evaluation_result.json"
    extraction_agent_model_name = "gpt-4o-mini"
    is_synthesis_evaluation = True
    llm = LLM(model="o3-mini")

    How It Works¶

    graph TB
    A[Ground Truth] --> C[LLM Agent]
    B[Test Data] --> C
    C --> D[Contextual Analysis]
    D --> E[Reasoning]
    E --> F[Scoring]
    F --> G[Evaluation Results]

    Custom LLM¶

    from comproscanner import evaluate_agentic
from crewai import LLM

# Use GPT-4
gpt4_llm = LLM(
    model="gpt-4-turbo",
    temperature=0.5,
    max_tokens=4096
)

results = evaluate_agentic(
    ground_truth_file="ground_truth.json",
    test_data_file="test_data.json",
    llm=gpt4_llm
)
